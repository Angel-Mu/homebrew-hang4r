# homebrew-hang4r

Homebrew tap for [hang4r](https://hang4r.dev) — the free macOS agents window
that runs Claude Code, Codex, and Cursor CLI agents in parallel.

```sh
brew tap angel-mu/hang4r
brew install --cask hang4r
```

Builds are Developer-ID signed and notarized. Releases:
[Angel-Mu/hang4r-releases](https://github.com/Angel-Mu/hang4r-releases/releases).
